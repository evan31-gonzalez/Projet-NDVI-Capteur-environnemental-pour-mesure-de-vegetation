# Projet-NDVI-Capteur-environnemental-pour-mesure-de-vegetation
Projet NDVI : conception et développement d’un capteur environnemental permettant d’évaluer l’état de la végétation à partir de mesures spectrales. Le projet inclut la modélisation 3D, la conception de PCB, l’électronique, le code embarqué et l’exploitation des données.

# Exports 3D

Ce dossier contient les fichiers exportés à partir de la modélisation Fusion 360.

Ces fichiers permettent d’utiliser le boîtier sans modifier le modèle source.

---

## Contenu du dossier

- `.stl` : fichier destiné à l’impression 3D
- `.step` : fichier destiné à l’assemblage mécanique (CAO) et au partage industriel

---

## Versionnement

Chaque export correspond à une version du boîtier.
Les fichiers sont nommés selon la convention :
boitier_ndvi_vX.Y.stl
boitier_ndvi_vX.Y.step

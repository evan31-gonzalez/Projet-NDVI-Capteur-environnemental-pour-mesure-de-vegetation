# Images du boîtier 3D

Ce dossier contient des images issues de la modélisation 3D du boîtier du capteur NDVI.

Les images permettent de visualiser :
- la géométrie générale du boîtier,
- l’intégration des composants,
- les vues extérieures et intérieures.

Ces images sont utilisées à des fins de documentation et de présentation.

# Fichiers Fusion 360

Ce dossier contient les fichiers sources Fusion 360 du boîtier du capteur NDVI.

Les fichiers `.f3d` sont les fichiers modifiables permettant de mettre à jour :
- la géométrie du boîtier,
- les supports de fixation,
- les ouvertures (capteurs, câbles, carte SD),
- l’intégration des cartes électroniques.

---

## Versionnement

Les fichiers sont nommés selon la convention :
boitier_ndvi_vX.Y.f3d

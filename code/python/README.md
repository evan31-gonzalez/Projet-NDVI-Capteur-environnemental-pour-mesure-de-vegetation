# Scripts Python – Traitement et visualisation NDVI

Ce dossier contient les scripts Python utilisés pour l’exploitation des données enregistrées par le capteur NDVI.

Ces scripts permettent :
- la lecture des fichiers de données issus de la carte SD,
- le traitement des mesures spectrales,
- la visualisation des données et de courbes (NDVI en fonction du temps).

---

## Organisation des scripts

Chaque version du script est conservée afin de suivre l’évolution du traitement des données.

Les fichiers sont nommés selon la convention :
visualisation_ndvi_vX.Y.py

---

## Version actuelle

- v0.2 : version permettant l’affichage des données et la génération des courbes NDVI depuis un fichier .csv.

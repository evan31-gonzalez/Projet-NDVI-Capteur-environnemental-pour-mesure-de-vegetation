# Exports du PCB

Ce dossier contient les fichiers d’export du circuit imprimé (PCB) au format image.

Ces fichiers sont générés à partir des outils de conception électronique et permettent de visualiser le schéma et le routage sans utiliser les fichiers sources.

---

## Contenu du dossier

Actuellement, ce dossier contient :
- le schéma électronique exporté au format PNG,
- la vue du routage côté top (Top View) au format PNG,
- la vue du routage côté bottom (Bottom View) au format PNG.

---

## Utilisation

Ces fichiers sont utilisés pour :
- la relecture du schéma électronique,
- la visualisation du routage du PCB,
- le partage avec les encadrants du projet,
- la documentation de l’avancement du design électronique.

---

## Remarque

Les fichiers sources modifiables du PCB (.sch, .brd) sont stockés dans le dossier principal `pcb/`.  
Ce dossier contient uniquement des fichiers d’export destinés à la consultation.

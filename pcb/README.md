# Carte électronique (PCB)

Ce dossier contient les fichiers de conception de la carte électronique du capteur NDVI.

La carte est conçue sous Eagle et regroupe :
- les deux capteurs spectraux AS7265x (vigne / ciel),
- l’Arduino Nano,
- le multiplexeur I2C TCA9548A,
- le capteur BME280,
- le MPU6050,
- le module RTC DS3231,
- le lecteur de carte SD.

Les fichiers .sch et .brd doivent toujours être utilisés ensemble.
